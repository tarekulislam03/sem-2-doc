##  Getting Started (First Time Only)

Run this command to clone this repo to your local pc.
```bash
git clone https://github.com/tarekulislam03/sem-2-doc.git
```


##  How to Contribute

If you have new documents to share, follow these steps:

1.  **Add** your files to the appropriate folder.
2.  **Commit** your changes:
    ```bash
    git add .
    git commit -m "Added [Subject/Topic] notes"
    ```
3.  **Push** to the repository:
    ```bash
    git push origin main
    ```
